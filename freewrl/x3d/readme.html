What is X3D compatibility, and what is X3D?

X3D is the follow on for VRML, and is being defined by the Web3D Consortium.

Don Brutzman of the Naval Postgraduate School in Monterey, California, created a template that allows an XML tool to translate X3D to VRML. We use this template to do our conversions.

Building and using X3D Compatibility.

Making the software:

X3D Compatibility is built in. However, it does need the following:

  1. perl module XML-LibXML-0.99 or later.
  2. perl module XML-LibXSLT-1.09 or later.
  3. Gnome Lib libxml2-2.3.10 or later.
  4. Gnome Lib libxslt-0.11.0 or later.

All of these modules/libraries are on the freewrl download site, here.

Installing the software:

  1. Uncompress and untar the library files, and cd into each directory in turn. The following steps will need to be acomplished, as root:
    • ./configure
    • make
    • make install
  2. run the ldconfig command, without parameters. eg, "ldconfig".
  3. Uncompress and untar the Perl XML* files. cd into each directory, and do the following to make and install these modules:
    • perl Makefile.PL
    • make install

Testing the software:

  1. run any of the numbered .xml tests in the "tests" directory.

Further Notes:

  1. The translation process needs ".dtd" files, as referenced in the .xml files. The first reference is for a web reference; the second is a reference to the local file system.

    If, like me, you have an http proxy in place, the local file will be referenced. You will have to ensure that the path in these files is correct by either:

    • editing the path in the .xml files
    • putting in symbolic links so that the files can be found in this directory. (they are included here)

    An example of the type of error you are likely to see is:

    "java.io.FileNotFoundException: /www.web3D.org/TaskGroups/x3d/translation/x3d-compact.dtd"

    One (not totally satisfactory) solution is to go as root:

    • cd /
    • ln -s . C:
    • ln -s . www.web3D.org
    • ln -s . TaskGroups
    • ln -s . x3d
    • ln -s /usr/local/src/FreeWRL-whatever-it-is/x3d translation
